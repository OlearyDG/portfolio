This is my final project for AP Computer Science 2018-2019. It is a version of Pac-Man made in Processing. It is a version of Pac-Man done in processing. A difficulty I had with this project was the ghost's movement, and to get it I use Math.random to have the ghosts move and reset it every time they touched the wall. But this created an issue where the ghosts would reach the outer border and eternally circle it never reaching the middle. to solve this I had it so every random interval the ghost randomly changes direction so the movement is more varied and it will stay in the middle of the maze. WHen converting it to Javascript to put it on this site one of the main issues was that JS doesn't let you disregard brackets if your command is only one line after an if statement, for loop, etc., so I had to manually add hundreds off brackets and there were errors with extra brackets and brackets connecting to the wrong set of brackets. The other issues were changing the key commands to work with JS and adding ".this" to all the classes.
